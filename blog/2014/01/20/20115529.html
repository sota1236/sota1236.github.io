* gistコマンド - アカウント設定

Jan 20 2014

gistをコマンドラインから投稿できるように設定した。

前からやろうと思ってたけど、めんどくてやってなかったがやったら簡単だった。

とりあえずメモメモ。

 

１．gistコマンドをインストール

みんなだいすきHomebrew. 

brew install gist

 

２．ユーザ登録

gist --login

上記のようにコマンドを打つとアカウント名とパスワードを聞かれるので入力。 

GitHub username: sota1235
              GitHub password: password
              Success! https://github.com/settings/applications
これだけ。簡単！

試しに投稿。 

% gist -f hello.js
              (type a gist.  to cancel,  when done)
              document.write("hello gist");
              [ctrl+d]

-fオプションでファイル名を指定できる。

gistページに行ったら見れたよ！

https://gist.github.com/8513912

ちなみにファイルをそのままアップロードしたかったらこう。 

gist hello.js
ファイル名を変えたい場合はこう。 
gist hello.js < upload.js
調べたらいろいろ出てきたけど結局こんだけで済むお話でした。

ちなみに。。。

アカウント設定しなくてもgistコマンドは使えますがanonymousで登録されます。

失敗したやつ

あとはアカウント設定した後でも-aコマンドをつければanonymous投稿できます。 

gist -a hello.js
